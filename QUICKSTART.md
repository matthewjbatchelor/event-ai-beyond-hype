# Quick Start Guide

## View the Website

### Method 1: Simple (Direct Open)
Double-click `index.html` to open it in your browser.

### Method 2: Local Server (Recommended)
Run this command in the terminal:
```bash
./start-server.sh
```

Or manually:
```bash
python3 -m http.server 8000
```

Then open: `http://localhost:8000`
